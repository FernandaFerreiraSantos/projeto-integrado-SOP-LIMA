FPOO

O que é?

Fundamentos de Programação Orientada a Objetos é uma disciplina que foca no uso de linguagens de programação orientadas a objetos para desenvolvimento. Neste caso, utilizamos Java, que é uma das linguagens mais utilizadas por empresas hoje em dia no desenvolvimento de WEB e Aplicações Móveis.

Programação Orientada a Objetos

O que é desenvolvido na linguagem Java usa classes e objetos. Entender esse conceito é a base para o uso da linguagem. Cada classe em Java pode ter atributos e métodos. Esta classe será usada como um modelo para poder criar vários objetos que usarão as propriedades e métodos fornecidos por esta classe em particular.

LIMA

O que é?

Uma linguagem de marcação é uma linguagem de computador que usa tags para definir elementos em um documento. É legível por humanos, o que significa que o arquivo de marcação contém palavras padrão em vez da sintaxe de programação típica. Embora existam muitas linguagens de marcação, as duas mais populares são HTML e XML.

HTML

Existem muitos tipos e padrões de linguagens de marcação, HTML (Hypertext Markup Language) é apenas um deles, mas com o advento da Internet, tornou-se popular porque a maioria dos documentos espalhados na Internet a utilizam para exibir informações. Outro exemplo de uma linguagem de marcação é chamada XML (Extensible Markup Language), que se diferencia do HTML por focar na construção de dados a serem enviados entre dois pontos de um mesmo sistema, e sua estrutura é escrita de forma mais restrita.

SOP

O que é?

Sistemas Operacionais é uma disciplina voltada ao conhecimento da parte não-física de um dispositivo, na qual é o softare responsável pela comunicação entre o hardware, ensinando conhecimentos sobre tipos de sistemas operacionais, instalação, maquinas virtuais e softwares desses sistemas.

Uma MV

Uma máquina virtual ou máquina virtual é um modelo de virtualização que cria uma abstração de recursos de computação e divide o computador em várias imagens de simulação. Em outras palavras, permite ao usuário acessar outra simulação interna de computador.

Instalação do Ubuntu

Ubuntu é uma distribuição Linux produzida pela empresa africana Canonical. Ele é um sistema operacional completo que pode ser instalado em computadores PC e Mac. O sistema é um dos Linux mais populares da atualidade.

Windows XP

O Windows XP é um sistema operacional da Microsoft para computadores, desktop, notebooks, tablets e media centers. Lançado em 2001, o sistema foi muito bem recebido pelos usuários e continua sendo muito utilizado, mesmo após o encerramento do suporte em abril de 2014.

HARE

O que é?

A Hardware and Networks está sempre buscando fornecer conhecimento sobre como as redes funcionam, tipos de redes, computadores montados e como identificar seus componentes.

Hardware

É a parte física do computador, ou seja, peças e equipamentos que fazem o computador funcionar. O termo também se refere ao conjunto de equipamentos acoplados em produtos que necessitam de gerenciamento computacional. Um dos hardwares mais populares são os processadores.

Redes

Uma rede de computadores pode ser definida como um grupo de dispositivos que podem trocar informações entre si, além de compartilhar os mesmos recursos.
